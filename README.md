## 1. НАЗВАНИЕ ПРОЕКТА

**«ProЗож: здоровье и спорт для каждого»**  

## 2. ДЛЯ КОГО ЭТОТ САЙТ

Сайт предназначен для людей, интересующихся здоровым образом жизни, правильным питанием и физической активностью.  
Целевая аудитория:  
- новички, которые хотят начать заниматься спортом, но не знают с чего;  
- любители фитнеса, бега, йоги, домашних тренировок;  
- люди, следящие за питанием и ищущие ПП-рецепты;  
- те, кто хочет читать мотивирующие статьи и научно-популярные материалы о здоровье.  

Возраст: от 16 до 45 лет. Уровень технической подготовки — любой (сайт должен быть простым и понятным).

## 3. КАКИЕ СТРАНИЦЫ БУДУТ НА САЙТЕ

1. **Главная**  
   — приветственный блок с девизом сайта;  
   — анонсы последних 3–5 статей;  
   — быстрые ссылки на популярные разделы (питание, тренировки, мотивация);  
   — форма подписки на новости (email).  

2. **Статьи / Блог**  
   — список всех статей с картинками, заголовками, датами и кратким описанием;  
   — поиск по ключевым словам;  
   — фильтр по категориям (питание, тренировки, психология, здоровье).  

3. **Питание**  
   — рецепты здоровых блюд (завтраки, обеды, ужины, перекусы);  
   — таблицы калорийности и БЖУ (базовые);  
   — советы по составлению рациона.  

4. **Тренировки**  
   — готовые программы тренировок (для дома, зала, на улице);  
   — видео или GIF-инструкции упражнений;  
   — раздел «тренировка дня».  

5. **Мотивация и ЗОЖ**  
   — истории успеха реальных людей;  
   — подборки полезных привычек;  
   — мифы о ЗОЖ и спорте (разоблачения).  

6. **Контакты / О проекте**  
   — форма обратной связи (имя, email, сообщение);  
   — ссылки на соцсети (Telegram, Instagram, VK);  
   — информация об авторе блога.  

## 4. ЧТО УМЕЕТ ДЕЛАТЬ САЙТ

- Показывать статьи с удобным чтением (адаптивный дизайн под телефон/планшет/ПК).  
- Иметь меню-навигацию по разделам.  
- Позволять искать статьи по ключевым словам.  
- Отображать форму подписки на новости (сохранение email в базу).  
- Принимать сообщения через форму обратной связи.  
- (можно добавить) Комментирование статей для зарегистрированных пользователей.  
- Быть быстрым и понятным даже на медленном интернете.  

## 5. ТЕХНОЛОГИИ
- **LAMP (Linux + Apache + MySQL + PHP)** — серверная платформа, на которой сайт будет работать.  
- **WordPress** — система управления содержимым (CMS), чтобы удобно добавлять статьи без программирования.  
- **GitHub** — хранение кода, ТЗ, диаграмм и возможных доработок.  
- HTML5, CSS3, JavaScript — для внешнего вида и интерактива.
